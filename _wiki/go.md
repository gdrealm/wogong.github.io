## install
simple, just download the tarball and set PATH.

<https://golang.org/doc/install#tarball>

    wget file
    tar xvf file
    
    export GOROOT=$HOME/go
    export GOPATH=$HOME/gowork # work directory
    export PATH=$PATH:$GOROOT/bin
