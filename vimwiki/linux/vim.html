  • 使用 vimwiki 提示错误： Can't open file syntax/AUTOMAKE.vim

在 .vim/syntax/ 路径下添加 AUTOMAKE.vim 文件即可解决。文件地址[http://vim.cybermirror.org/runtime/ftplugin/automake.vim AUTOMAKE.vim]
