DNS: Domain Name Server

目前使用[[DNSpod|http://dnspod.cn]], 推荐。

如何测试DNS服务器：dig

dig @166.111.8.28 wogong.net

清除DNS缓存

  1. Windows

    ipconfig/flushdns
net stop/start dnscache

  2. Mac

    sudo killall -HUP mDNSResponder (for Lion & Mountain Lion)
