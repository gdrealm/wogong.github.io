== 常用命令 ==

删除邮件队列中全部文件：

  • postsuper -d ALL 删除队列中全部邮件

  • postsuper -d ID 删除指定ID邮件

显示邮件队列邮件 postcat -q ID

postmap hash:/etc/postfix/name 大部分文件都需要这样做才OK

/etc/aliases 设置别名

修改之后运行 {{{sudo newaliases}}}生效。
